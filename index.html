EventStack:

Website for collecting entries of students. In this website, every user has a profile section which contains all the necessary information about the user like personal inforamation, academic information passport, and signature photo. The organizer can create an empty table by selecting columns from the list of the predefined set of columns. After creating table organizer sends it to students. Student gets invitation of that event. When Student accepts it, all the details of that student are fetched from the profile section and filled in that table. This can help students from filling the same information again and again for different events. The table can be downloaded in excel sheet also.

Depository:

Online portal for reporting lost items in the college campus. The user can report lost items to the portal. A search panel is also there for searching lost items. This Website helps in finding lost items. When a person finds any item in the campus then he/she can post the item on it with details like where it found, its identification marks. So whenever a user searches on portal using details like item type and location where he/she had lost, all matching item will be displayed, and the user can identify own item and contact respective person using contact number or email.

Picnet:

Online photo sharing website. The user can create an album and add friends in it. Each added friend can upload photos, view photos and download selected photos(with original quality) to that album. This makes photo sharing easy and keeps photo separated album-wise.

Order-Book:

Order management web app for printing press shop. The operator can create a new firm account, add new orders to it and can update the status of an order. It have separate tabs for pending orders, due payment orders. A login credential is also provided to the operator.
